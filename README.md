# Interior Design – Short-Term Rental | São Paulo

Interior design project located in São Paulo, near the Faria Lima area,
developed for short-term rental use (Airbnb and similar platforms).

The scope includes complete interior planning, covering custom millwork,
flooring design, layout coordination, material specification, and surface
finishes. The project focuses on durability, easy maintenance, and visual
consistency, balancing aesthetic appeal with practical use for high turnover
occupancy.

## Scope
- Interior design
- Custom millwork
- Flooring and layout coordination
- Material and finish specification

## Use
Short-term rental (Airbnb)

## Location
São Paulo, Brazil
